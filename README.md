# Right Hand or Left Hand

## 🖼️ Image Gallery

Welcome to the image gallery! This repository showcases images in a beautiful, social-media style layout.

### Getting Started

1. **Enable GitHub Pages**:
   - Go to Settings → Pages
   - Set source to "Deploy from a branch"
   - Select `main` branch
   - Your gallery will be live at: `https://cookc227-ctrl-alt.github.io/right-hand-or-left-hand`

2. **Add Your Images**:
   - Create an `images/` folder in this repository
   - Upload your images (JPG, PNG, WebP, etc.)
   - Images will automatically appear in the gallery

3. **Customize**:
   - Edit `index.html` to change titles, descriptions, and styling
   - Modify the color scheme by updating the CSS gradient colors
   - Add captions for each image

### Features

✨ **Responsive Design** - Works on desktop, tablet, and mobile  
🎨 **Beautiful UI** - Modern gradient background with smooth animations  
🖱️ **Hover Effects** - Interactive cards with overlay information  
📱 **Social Media Style** - Grid layout similar to Instagram or Pinterest  
🔍 **Full Screen Viewer** - Click images to view in full screen  
⚡ **Fast Loading** - Static HTML with no external dependencies  

### File Structure

```
right-hand-or-left-hand/
├── index.html          # Main gallery page
├── README.md           # This file
└── images/             # Your image folder
    ├── image-1.jpg
    ├── image-2.jpg
    └── ...
```

### Tips

- Use images with consistent aspect ratios for the best look
- Optimize images (compress) for faster loading
- Supported formats: JPG, PNG, WebP, GIF
- Images will automatically fill a square grid layout
